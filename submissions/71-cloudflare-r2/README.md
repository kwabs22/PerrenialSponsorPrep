# Cloudflare R2 Storage

**Showcases:** Cloudflare R2 for zero-egress object storage

## What it does
Store objects with Cloudflare R2's S3-compatible API. Zero egress fees make it perfect for serving assets globally.

## Latest Feature (December 2024)
- **Zero Egress** - No bandwidth charges
- **S3 Compatible** - Drop-in replacement
- **Edge Integration** - Workers access
- **Event Notifications** - Queue triggers

## Setup

1. Install: `pip install -r requirements.txt`
2. Copy `.env.example` to `.env`
3. Run: `python main.py`
