# Twilio Multi-Channel Verify

**Showcases:** Verify v2 + WhatsApp

## What it does
Multi-channel verification supporting SMS, WhatsApp, and Voice using Twilio Verify v2.

## Latest Feature (2024)
- **Verify v2** - Enhanced verification API
- **WhatsApp Templates** - Business messaging
- **Silent Network Auth** - Invisible verification

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
