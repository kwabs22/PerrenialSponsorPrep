# Slack Standup Bot

**Showcases:** Workflow Builder API

## What it does
An automated standup bot that collects and summarizes daily updates using Slack's Workflow Builder API.

## Latest Feature (2024)
- **Workflow Builder API** - Custom workflow steps
- **Block Kit** - Rich message formatting
- **Scheduled Messages** - Timed delivery

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
