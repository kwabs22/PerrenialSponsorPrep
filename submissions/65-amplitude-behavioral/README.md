# Amplitude Behavioral Analytics

**Showcases:** Amplitude for behavioral analytics and AI insights

## What it does
Understand user behavior with Amplitude's self-serve analytics platform. Build charts, analyze user paths, and get AI-powered insights without writing SQL.

## Latest Feature (December 2024)
- **AI Insights** - Automatic anomaly detection
- **Session Replay** - Visual behavior debugging
- **Experiment** - Built-in A/B testing
- **CDP** - Customer data platform features

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env` and add your API key:
   ```bash
   cp .env.example .env
   ```

3. Run the demo:
   ```bash
   python main.py
   ```
