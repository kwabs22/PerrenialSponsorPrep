# MongoDB Vector Search Recommendations

**Showcases:** Atlas Vector Search

## What it does
AI-powered product recommendations using MongoDB's native vector search capabilities.

## Latest Feature (2024)
- **Atlas Vector Search** - Native vector search in MongoDB
- Seamless integration with aggregation pipeline
- Pre-filtering with metadata

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
