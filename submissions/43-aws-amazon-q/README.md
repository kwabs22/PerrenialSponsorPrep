# AWS Amazon Q Data Pipeline

**Showcases:** Amazon Q + Bedrock

## What it does
AI-powered data pipeline assistant using Amazon Q for code generation and analysis.

## Latest Feature (2024)
- **Amazon Q** - AI coding assistant
- **Bedrock** - Foundation models
- **CodeWhisperer** - Code suggestions

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
