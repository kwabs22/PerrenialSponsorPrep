# PostHog Product Analytics

**Showcases:** PostHog Events, Feature Flags, Session Replay

## What it does
Implement product analytics for your hackathon project with PostHog's all-in-one platform. Track user events, run A/B tests with feature flags, and debug issues with session replay.

## Latest Feature (December 2024)
- **Session Replay** - Visual debugging of user sessions
- **Feature Flags** - A/B testing and gradual rollouts
- **AI Analytics** - Natural language queries (HogQL)
- **CDP Integration** - Customer data platform features

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env` and add your API key:
   ```bash
   cp .env.example .env
   ```

3. Run the demo:
   ```bash
   python main.py
   ```
