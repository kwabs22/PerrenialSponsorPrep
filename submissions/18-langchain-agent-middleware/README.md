# LangChain Agent with Middleware

**Showcases:** create_agent with Middleware System (v1.1)

## What it does
An AI agent with built-in PII redaction middleware that automatically sanitizes sensitive information.

## Latest Feature (2024)
- **create_agent** - New standard agent API
- **Middleware System** - Customizable agent behavior
- Built-in middlewares: PII, Summarization, Human-in-the-Loop

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
