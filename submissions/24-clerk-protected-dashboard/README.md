# Clerk Protected Dashboard

**Showcases:** React Router SDK + Reverification

## What it does
A protected admin dashboard with Clerk authentication, featuring the new reverification flow for sensitive actions.

## Latest Feature (December 2024)
- **React Router SDK** - Native routing integration
- **Reverification** - Confirm identity for sensitive actions
- Combined sign-in/sign-up flow

## Setup
```bash
npm install
cp .env.example .env
npm run dev
```
