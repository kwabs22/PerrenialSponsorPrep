# CrewAI Research Crew

**Showcases:** Multi-Agent Crews

## What it does
A team of AI agents that collaborate to research any topic - one searches, one analyzes, one writes.

## Latest Feature (2024)
- **Multi-agent Crews** - Agents working together
- Role-based task delegation
- Hierarchical process management

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py "artificial intelligence"
```
