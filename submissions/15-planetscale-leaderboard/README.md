# PlanetScale High-Performance Leaderboard

**Showcases:** Boost Query Caching

## What it does
A high-performance gaming leaderboard using PlanetScale's Boost for sub-millisecond queries.

## Latest Feature (2024)
- **Boost** - Automatic query caching
- Serverless MySQL at the edge
- Branching for schema changes

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
