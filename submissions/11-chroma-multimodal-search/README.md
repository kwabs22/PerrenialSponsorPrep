# Chroma Multimodal Search

**Showcases:** Multi-modal Embeddings

## What it does
Search across images and text together using Chroma's multimodal embedding support.

## Latest Feature (2024)
- **Multi-modal embeddings** - CLIP-style image+text search
- Local-first with optional cloud sync
- Simple Python API

## Setup
```bash
pip install -r requirements.txt
python main.py
```
