# AI21 Jamba

**Showcases:** AI21 Jamba for hybrid SSM-Transformer architecture

## What it does
Use AI21's Jamba model - combining SSM and Transformer for long context at high throughput.

## Latest Feature (December 2024)
- **Jamba 1.5** - Hybrid architecture
- **256K Context** - Massive context window
- **Task-Specific Models** - Summarization, paraphrasing

## Setup
```bash
pip install -r requirements.txt && python main.py
```
