# Weights & Biases Experiment Tracking

**Showcases:** W&B for ML experiment tracking and LLM observability

## What it does
Track experiments, visualize metrics, and debug ML models with Weights & Biases. Includes Weave for LLM application tracing and prompt management.

## Latest Feature (December 2024)
- **Weave** - LLM application observability
- **Prompts** - Prompt versioning and management
- **Tables** - Data visualization and comparison
- **Sweeps** - Hyperparameter optimization

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Login to W&B:
   ```bash
   wandb login
   ```

3. Run the demo:
   ```bash
   python main.py
   ```
