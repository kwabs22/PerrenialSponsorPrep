# Uploadcare File Upload

**Showcases:** Uploadcare for file uploads and CDN delivery

## What it does
Handle file uploads with Uploadcare's widget and API. Smart CDN, image transformations, and secure delivery built-in.

## Latest Feature (December 2024)
- **Smart CDN** - Automatic optimization
- **File Uploader** - Drop-in widget
- **Transformations** - URL-based editing
- **Secure Delivery** - Signed URLs

## Setup

1. Install: `pip install -r requirements.txt`
2. Copy `.env.example` to `.env`
3. Run: `python main.py`
