# Backblaze B2 Storage

**Showcases:** Backblaze B2 for S3-compatible object storage

## What it does
Store and serve files with Backblaze B2's affordable S3-compatible API. 1/5th the cost of AWS S3 with the same interface.

## Latest Feature (December 2024)
- **S3 Compatible** - Use existing S3 SDKs
- **Free Egress** - To Cloudflare, Fastly
- **Object Lock** - Immutable backups
- **Event Notifications** - Webhooks

## Setup

1. Install: `pip install -r requirements.txt`
2. Copy `.env.example` to `.env`
3. Run: `python main.py`
