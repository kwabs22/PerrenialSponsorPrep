# Supabase Cron Notifications

**Showcases:** Edge Functions + pg_cron

## What it does
A scheduled notification system using Supabase's new Cron Jobs feature to send automated alerts.

## Latest Feature (2024)
- **pg_cron** - Schedule PostgreSQL jobs
- **Edge Functions** - Deploy serverless functions
- Built-in dashboard for function management

## Setup
```bash
npm install
cp .env.example .env
npx supabase functions deploy
```
