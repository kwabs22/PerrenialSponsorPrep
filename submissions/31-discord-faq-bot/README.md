# Discord FAQ Bot

**Showcases:** Application Commands (Slash Commands)

## What it does
An AI-powered FAQ bot using Discord's Application Commands for natural question answering.

## Latest Feature (2024)
- **Application Commands** - Native slash commands
- **Autocomplete** - Real-time suggestions
- **Components** - Buttons and select menus

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
