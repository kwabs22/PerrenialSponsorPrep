# Gradio Model Comparison

**Showcases:** Gradio 5 + SSR

## What it does
A side-by-side model comparison interface using Gradio 5's new features.

## Latest Feature (2024)
- **Gradio 5** - Major performance improvements
- **SSR** - Server-side rendering for faster load
- **New components** - Enhanced UI elements

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
