# Stripe Subscriptions with Entitlements

**Showcases:** Adaptive Pricing + Entitlements API

## What it does
A subscription system with feature flags using Stripe's new Entitlements API.

## Latest Feature (2024)
- **Entitlements API** - Feature access control
- **Adaptive Pricing** - Dynamic regional pricing
- Date-based API versioning

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
