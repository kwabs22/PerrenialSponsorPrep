# Google Cloud Vertex AI Chatbot

**Showcases:** Vertex AI + Gemini 2.0

## What it does
Enterprise chatbot using Google Cloud's Vertex AI with Gemini 2.0.

## Latest Feature (2024)
- **Gemini 2.0** - Latest multimodal model
- **Vertex AI** - Enterprise ML platform
- **Grounding** - Search-grounded responses

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
