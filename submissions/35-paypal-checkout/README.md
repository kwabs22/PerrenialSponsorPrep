# PayPal Advanced Checkout

**Showcases:** Advanced Checkout + Card Fields

## What it does
Custom checkout experience with PayPal's Advanced Checkout and hosted card fields.

## Latest Feature (2024)
- **Advanced Checkout** - Full customization
- **Card Fields** - PCI-compliant hosted inputs
- **Pay Later** - BNPL options

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
