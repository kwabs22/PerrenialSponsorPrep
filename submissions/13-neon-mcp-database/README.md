# Neon MCP Database Manager

**Showcases:** MCP Server for AI Agents

## What it does
Manage your Neon Postgres database using natural language via Claude or other AI agents with the Model Context Protocol.

## Latest Feature (December 2024)
- **MCP Server** - Let AI agents manage your database
- Natural language SQL queries
- Branch management via chat
- Schema migrations with AI assistance

## Setup
```bash
npm install @neondatabase/mcp-server-neon
# Configure in Claude Desktop or other MCP client
```
