# Azure OpenAI Realtime Voice

**Showcases:** gpt-4o-realtime + Voice Translation

## What it does
Real-time voice translation using Azure OpenAI's gpt-4o-realtime-preview.

## Latest Feature (2024)
- **gpt-4o-realtime** - Low-latency voice AI
- **Voice Translation** - Multi-language support
- **Azure AI Speech** - Enhanced TTS/STT

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
