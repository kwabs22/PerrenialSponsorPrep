# Plaid Bank Verification

**Showcases:** Layer + Beacon

## What it does
Bank account verification with fraud detection using Plaid's Layer and Beacon APIs.

## Latest Feature (2024)
- **Layer** - One-click account verification
- **Beacon** - Fraud detection network
- **Identity Verification** - Enhanced KYC

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
