# Weaviate AI Product Catalog

**Showcases:** Generative Search Modules

## What it does
An AI-powered product catalog with generative search. Ask questions about products in natural language and get AI-generated answers with relevant product recommendations.

## Latest Feature (2024)
- **Generative Search** - RAG built into the database
- Multiple AI module integrations (OpenAI, Cohere, etc.)
- Hybrid search combining vector + keyword
- GraphQL API for flexible queries

## Setup

```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
