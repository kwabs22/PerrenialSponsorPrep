# Haystack Document Pipeline

**Showcases:** Pipeline as Code + Haystack 2.0

## What it does
A document processing pipeline that extracts, transforms, and indexes documents using Haystack's declarative pipeline syntax.

## Latest Feature (2024)
- **Pipeline as Code** - Declarative YAML/Python pipelines
- **Haystack 2.0** - New component architecture
- **Improved document stores**

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
