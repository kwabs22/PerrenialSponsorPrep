# LlamaIndex Knowledge Graph Builder

**Showcases:** Workflows + Property Graph Index

## What it does
Builds a knowledge graph from documents using LlamaIndex's new Property Graph Index and Workflows.

## Latest Feature (2024)
- **Workflows** - Declarative multi-step pipelines
- **Property Graph Index** - Entity-relation knowledge graphs
- **Improved RAG** - Graph-enhanced retrieval

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
