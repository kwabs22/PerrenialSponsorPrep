# AutoGen Game Agent

**Showcases:** AG2 + StateFlow

## What it does
A multi-agent game where AI agents play a collaborative word game using AutoGen's new StateFlow.

## Latest Feature (2024)
- **AG2** - AutoGen 2.0 architecture
- **StateFlow** - State machine for agent orchestration
- **Improved conversation patterns**

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
