# Segment Customer Data Platform

**Showcases:** Segment for customer data collection and routing

## What it does
Collect, unify, and activate customer data with Segment. Send events from any source to any destination with a single API - perfect for analytics, marketing, and personalization.

## Latest Feature (December 2024)
- **Profiles** - Unified customer profiles
- **Audiences** - Real-time segmentation
- **Journeys** - Automated workflows
- **Reverse ETL** - Warehouse to tools sync

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env` and add your write key:
   ```bash
   cp .env.example .env
   ```

3. Run the demo:
   ```bash
   python main.py
   ```
