# ImageKit Image Optimization

**Showcases:** ImageKit for real-time image optimization and delivery

## What it does
Optimize, transform, and deliver images with ImageKit's URL-based API. Automatic format selection, responsive images, and AI-powered features.

## Latest Feature (December 2024)
- **AI Background Removal** - Remove/replace backgrounds
- **Smart Crop** - AI-powered cropping
- **DAM** - Digital asset management
- **Real-time Transforms** - URL-based editing

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```

3. Run: `python main.py`
