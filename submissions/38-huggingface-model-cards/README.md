# HuggingFace Model Cards

**Showcases:** llms.txt + Hub v1.0

## What it does
Automated model documentation and card generation using HuggingFace Hub.

## Latest Feature (2024)
- **llms.txt** - Standardized model info
- **Hub v1.0** - Stable API
- **Model Cards** - Enhanced documentation

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
