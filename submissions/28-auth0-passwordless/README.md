# Auth0 Passwordless Demo

**Showcases:** Passkeys + ML Bot Detection

## What it does
A passwordless authentication flow using Auth0's passkey support and bot detection.

## Latest Feature (2024)
- **Passkeys** - WebAuthn-based authentication
- **ML Bot Detection** - Adaptive security
- **Universal Login** - Enhanced customization

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
