# Vercel AI Elements Chat

**Showcases:** AI SDK 5 + AI Elements

## What it does
A beautiful streaming chat UI built with Vercel's new AI Elements - prebuilt components for AI interfaces.

## Latest Feature (2024)
- **AI SDK 5** - Type-safe chat, agentic loops
- **AI Elements** - Prebuilt AI UI components
- Speech generation support

## Setup
```bash
npm install
cp .env.example .env
npm run dev
```
