# Mapbox Trip Planner

**Showcases:** Standard Maps + Navigation

## What it does
Interactive trip planning with Mapbox Standard maps and turn-by-turn navigation.

## Latest Feature (2024)
- **Standard Maps** - New map style
- **3D Terrain** - Elevation visualization
- **Navigation SDK** - Turn-by-turn guidance

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
