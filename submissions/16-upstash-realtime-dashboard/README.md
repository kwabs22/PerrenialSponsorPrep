# Upstash Real-time Activity Dashboard

**Showcases:** REST API with MONITOR via SSE

## What it does
A real-time dashboard that monitors Redis activity using Upstash's new Server-Sent Events support.

## Latest Feature (2024)
- **MONITOR via SSE** - Stream Redis commands in real-time
- **REST API** - No connection management needed
- IP Allowlist for security

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
