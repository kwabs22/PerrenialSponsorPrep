# Together AI Inference

**Showcases:** Together Inference API + Fine-tuning

## What it does
Access 100+ open-source models through Together AI's inference platform. Run Llama, Mistral, CodeLlama, and more with competitive pricing and fine-tuning capabilities.

## Latest Feature (December 2024)
- **Llama 3.2 Vision** - Multimodal inference
- **Fine-tuning API** - Custom model training
- **Function Calling** - Tool use for open models
- **Embeddings** - Text and code embeddings

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env` and add your API key:
   ```bash
   cp .env.example .env
   ```

3. Run the demo:
   ```bash
   python main.py
   ```
