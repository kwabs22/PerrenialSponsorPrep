# Cloudflare Edge Notes App

**Showcases:** Workers Builds + D1

## What it does
A notes app running entirely on the edge using Cloudflare Workers and D1 database.

## Latest Feature (2024)
- **Workers Builds** - CI/CD for Workers
- **D1** - Edge SQLite database
- **KV** - Global key-value store

## Setup
```bash
npm install
cp .env.example .env
npm run dev
```
