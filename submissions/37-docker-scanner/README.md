# Docker Security Scanner

**Showcases:** Scout Policy Evaluation

## What it does
Container security scanning with policy enforcement using Docker Scout.

## Latest Feature (2024)
- **Scout Policy Evaluation** - Custom security policies
- **SBOM Analysis** - Software bill of materials
- **CVE Database** - Vulnerability tracking

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```
