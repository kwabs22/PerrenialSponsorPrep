# Mux Video Streaming

**Showcases:** Mux for video streaming and real-time video

## What it does
Build video experiences with Mux's API-first platform. Upload, encode, and stream video with adaptive bitrate, plus real-time video for live applications.

## Latest Feature (December 2024)
- **Real-time Video** - WebRTC streaming
- **Data API** - Video analytics
- **AI Captions** - Auto-generated subtitles
- **Player SDK** - Customizable video player

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```

3. Run the demo:
   ```bash
   python main.py
   ```
