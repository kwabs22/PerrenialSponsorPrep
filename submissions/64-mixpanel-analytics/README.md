# Mixpanel Event Analytics

**Showcases:** Mixpanel for product analytics and user behavior tracking

## What it does
Track user behavior, analyze funnels, and measure retention with Mixpanel. Get insights into how users interact with your product through powerful analytics and cohort analysis.

## Latest Feature (December 2024)
- **Warehouse Connectors** - Sync with data warehouses
- **Session Replay** - Visual behavior analysis
- **AI Insights** - Automated anomaly detection
- **Group Analytics** - B2B account-level metrics

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env` and add your token:
   ```bash
   cp .env.example .env
   ```

3. Run the demo:
   ```bash
   python main.py
   ```
