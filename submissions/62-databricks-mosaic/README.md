# Databricks Mosaic AI

**Showcases:** Databricks Mosaic AI for enterprise generative AI

## What it does
Build production AI applications on Databricks with Mosaic AI. Access foundation models, vector search, and model serving - all integrated with your lakehouse data.

## Latest Feature (December 2024)
- **Mosaic AI Agent Framework** - Build AI agents
- **Vector Search** - Serverless similarity search
- **Model Serving** - Deploy any model
- **AI Gateway** - Unified API for multiple providers

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Copy `.env.example` to `.env` and add your token:
   ```bash
   cp .env.example .env
   ```

3. Run the demo:
   ```bash
   python main.py
   ```
