# Perplexity Sonar API

**Showcases:** Perplexity for AI-powered research with citations

## What it does
Access Perplexity's search-augmented LLM API. Get answers with real-time web citations - perfect for research assistants and fact-checked AI.

## Latest Feature (December 2024)
- **Sonar API** - Search-grounded responses
- **Citations** - Source attribution
- **Online Models** - Real-time web access
- **Pro Search** - Deep research mode

## Setup
```bash
pip install -r requirements.txt
cp .env.example .env  # Add PERPLEXITY_API_KEY
python main.py
```
