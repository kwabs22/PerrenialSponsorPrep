# Lightning AI Studios

**Showcases:** Lightning Studios + Lightning Serve for ML development

## What it does
Develop and deploy ML models with Lightning AI's cloud IDE and serving platform. Get instant access to GPUs, collaborative workspaces, and one-click deployment.

## Latest Feature (December 2024)
- **Studios** - Cloud IDE with GPU access
- **Serve** - Model deployment platform
- **Teams** - Collaborative workspaces
- **Apps** - Full-stack ML applications

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Login to Lightning:
   ```bash
   lightning login
   ```

3. Run the demo:
   ```bash
   python main.py
   ```
